# 夜航猫头鹰 IP

## 角色定义

夜航猫头鹰是正文漫画里的固定动作主体。它不是吉祥物、贴纸或装饰，也不是把画面改成某个品牌世界观的入口；它认真参与系统运转、文本判断、结构校准或交付检查。

如果去掉猫头鹰，图的核心隐喻还完全成立，说明角色太装饰；要重写提示词，让它成为动作主体。如果第一眼只看到猫头鹰，看不懂文章结构，说明角色太大或太复杂；要缩小角色并减少细节。

## 外形

- 猫头鹰主体，不是黑色实心旧角色。
- 暖橙色脸盘和耳羽。
- 蓝绿色头部和身体。
- 黑色大墨镜，墨镜下只露出少量白点高光或空洞眼神。
- 深蓝色宽松外袍。
- 白色咖啡杯，常在手里、桌边、箱子上或机器旁；它是角色随身物，不是内容隐喻。
- 细腿、简单脚，身体可以略圆、略笨重。
- 轮廓略微不规则，有手绘感。
- 羽毛细节少量即可，默认保留几笔短线，不要满身纹理。

## 默认复杂度

用户说“上一个味道好”“按这个形象”时，使用 `assets/nightsailing-owl-reference.png` 的复杂度：

- 保留明显的耳羽、脸盘、墨镜、蓝袍、咖啡杯。
- 允许少量羽毛短线和衣褶。
- 在正文配图里可以缩小角色，但不要压成过度简化的图标。

只有用户明确要求“更简化”时，才把耳羽变成更少笔画、袖子变成整块蓝形、删掉大部分羽毛线。

## 性格

- 冷静、审稿、看穿套路。
- 空表情，有点困但很清醒。
- 像一个低调的系统操作员或编辑。
- 冷幽默，不卖萌。
- 认真做一件荒诞但成立的事。

## 常见职责

让夜航猫头鹰承担核心动作：

- 操作旧机器、漏斗、秤、门、井、梯子、水管、纸箱、抽屉、黑盒、打孔器、压面机、扫描仪。
- 拿放大镜看需求。
- 给稿件盖章、删字、称重、打孔。
- 守门，决定内容能不能过。
- 把默认套路丢进废纸篓。
- 把输入纸条整理成可交付卡片。
- 在路线、漏斗、分层盒子、怪工位中承担关键动作。

动作库来自文章内容，不要因为角色叫 NightSailing 就自动加入船、海、灯塔、夜景、星月或航海物件。

## 禁止

- 不要画成黑色实心小怪物。
- 不要画成黑色实心角色戴猫头鹰耳朵。
- 不要画成儿童卡通或过度可爱 mascot。
- 不要给复杂服装、夸张表情包、闪亮眼睛。
- 不要让它只是站在角落里看。
- 不要让它抢走结构表达。
- 不要 3D、拟真、厚重阴影、复杂羽毛。
- 不要把“夜航”“NightSailing”“夜航船”写在图上。
- 不要把船、灯塔、码头、海浪、月亮、星空作为默认道具。
